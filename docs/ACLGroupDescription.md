# ACLGroupDescription Module
## Description

Wenn der Match Parameter nicht mitgegeben wird, dann werden alle Gruppen ausgegeben, welche nicht 端bereinstimmen.
Die Angabe von Searchbase, Pattern und Filter Parameter sind zwingend erforderlich.
